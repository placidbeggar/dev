CSS Selectors

  1. *:Selects absolutely all elements on the page. Weight:0
  2. .class:Selects an element with a class attribute. Weight:10
  3. #id:Selects an element with an id attribute. Weight:100
  4. [attr=val]:Selects elements that have an attribute with the specified value.Selector of attributes with the exact value. Weight:10
  5. A:Any HTML element for which formatting rules are defined can be a selector. Weight:1
  6. .class.class:Multiple classes can be added to any element at the same time. Weight:20
  7. inline-style:Gets a specificity value of 1000. Weight:1000
  8. !important:Rule in CSS is used to add more importance to a property/value than normal.Weight:10000
  9. #id #id2 a: Two id selectors and one type selector. Weight: 201
  10. :last-of-type:Pseudo-class.Selects the last element with the same tag. Weight:10
